💊 Medical・Pharma LLM Projects

  • Foundation models × Healthcare network data is a collaborative research with Kyoto Univ. and EQUES Inc. Ongoing.
  • JPharmatron-7B is the first Japanese pharmaceutical domain specific LLM, developed along with new benchamrks JPharmaBench. A work at EQUES Inc.
  • Japanese Medical Language Model Evaluation Harness is the evaluation codes and leaderboard for medical LLMs. A work at AIST KAKUSEI.
  • MedSwallow is the second Japanese medical LLM finetuned from Swallow-70b-instruct-hf. A work at UTokyo Hospital.
  • JMedLoRA is the first Japanese medical LLM finetuned from Llama2. A work at UTokyo Hospital.
  • awesome-latest-LLM is the curation of the latest and competent LLMs, including medical domain adaptation. Personal work.

🗒 Statistics Projects

  • Minimum information Markov model is a brand new Markov model that models the marginal stationary distribution and the dependence function seperately.
  • Sparse Torus Graph Modeling × EEG introduces a high-dimensional circular model with regularization to estimate dependence structure behind EEG phase, i.e., phase-based connectivity.
  • Minimum Information Copula is the library that numerically calculates the density function of minimum information copula, the most natural copula under predetermined constraints.
  • OGI (objective general index) implementation in Python is the Python implementation of Sei (2016).
